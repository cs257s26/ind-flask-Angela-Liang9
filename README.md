# README

Individual Flask project.
Configure the port with the variable PORT in app.py

Usage & Examples

To see water data for all locations for a given year, enter http://127.0.0.1:PORT/year/YEAR
For example: http://127.0.0.1:PORT/year/2005

To see water data for a specific location and year, enter http://127.0.0.1:PORT/LOCATION/YEAR
For example: http://127.0.0.1:PORT/France/2015
